# The Cabin

The cabin is a short first person horror game made in Unity, where you explore around dark cabin and its outdoor environment. Try walking around, use the flashlight for your visibility, hear scary audio and footsteps.

This was made for a portfolio project while learning Unity for the first time. The game is currently is a walking exploration only but i plan to add more something in the future.

All assets are free from the Unity Asset Store. Tools used are Unity, C#, and Visual Studio Code.

Controls of the game
- WASD = Move
- Mouse = Look around
- E = Interact with the door
- ESC = Back to main menu
